## Logo

Reemplaza `assets/logo.png` por el logo oficial.

Recomendado: PNG transparente, cuadrado, 512 × 512 px, menor a 150 KB si puedes. En la firma se muestra como 78 × 78 px.

Si tu logo es horizontal, usa PNG transparente de 600 × 200 px y cambia en los HTML:

`width="78" height="78"` por `width="105" height="35"`.

## Gmail

1. Abre la firma desde GitHub Pages.
2. Selecciona la firma visual renderizada, no el código.
3. Copia con Ctrl+C.
4. En Gmail: Settings → See all settings → General → Signature.
5. Crea una nueva firma y pega.
6. Configura Signature defaults para nuevos correos y respuestas.
7. Save Changes.
